# Asset Credits

This file records the source and license of every external art or audio asset
used in Shift Panic, as required by `AGENTS.md`.

## Current status

As of Milestone 8, the game uses **no external assets**. All visuals are drawn
procedurally in-engine (GDScript `_draw`, shaders, and `GPUParticles2D`) and all
audio is generated procedurally at runtime in `audio_feedback_controller.gd`.

When the first external CC0 / public-domain asset is added, list it here in the
table below.

| Asset | Type | Source | License |
| ----- | ---- | ------ | ------- |
| _(none yet)_ | | | |

## Rules

- External assets must be CC0 or public domain.
- Record the asset name, type, source URL, and license here before use.
- Prefer in-engine procedural art when it reaches the needed quality.
